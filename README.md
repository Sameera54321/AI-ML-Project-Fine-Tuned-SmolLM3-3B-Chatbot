# 🚀 Fine-Tuned SmolLM3-3B Chatbot

This project demonstrates the fine-tuning and deployment of the **SmolLM3-3B** language model using a custom conversational dataset. The model was trained using **Kaggle GPU (T4 x2)** and deployed publicly using **Hugging Face Spaces** with a **Gradio-based chatbot interface**.

---

## 📌 Project Overview

The goal of this project was to:
- Prepare a custom AI conversational dataset
- Fine-tune the SmolLM3-3B language model
- Train the model using Kaggle GPU resources
- Deploy the chatbot publicly for interactive use

This project provided hands-on experience with:
- Large Language Models (LLMs)
- AI model fine-tuning
- Dataset preparation
- Hugging Face deployment
- Gradio chatbot interfaces

## 📂 Dataset

A custom dataset was created in CSV format using conversational question-answer pairs.

## ⚡ Training Process

The model was fine-tuned on Kaggle using GPU acceleration with:

Hugging Face Transformers
TRL (Transformer Reinforcement Learning)
Custom conversational dataset

Training included:

Dataset loading
Tokenization
Fine-tuning for 3 epochs
Model evaluation
